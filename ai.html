How it works

어떻게 작동하는가

1

CLAUDE.md — 시스템 설계서

프로젝트 컨텍스트, 에이전트 라우팅 규칙, 지식베이스 참조 경로, 피드백 루프를 하나의 마크다운 파일에 정의합니다. Claude가 세션을 시작할 때 이 파일을 읽고 "비서실장"으로서의 역할과 판단 기준을 자동으로 갖춥니다.

2

커스텀 스킬 — 반복 업무 자동화

/content monitor 한 줄이면 내부·외부 소스를 스캔해 블로그 소재 5~10개를 제안합니다. /content draft로 SEO 최적화된 초안을 생성하고, /content publish로 Ghost CMS에 바로 발행합니다.

3

MCP — 외부 도구 실시간 연결

Notion에서 프로젝트 문서를 가져오고, Slack 대화를 검색하고, Google Calendar에 일정을 만듭니다. GA4 Data API로 트래픽·전환 데이터를 실시간 조회합니다. 별도 대시보드 없이 CLI 한 곳에서 모든 데이터에 접근합니다.

4

메모리 & 학습 — 피드백이 쌓인다

대화 중 사실 보정이나 실무 노하우를 감지하면 knowledge/에 자동 저장합니다. 같은 실수를 반복하지 않고, 회사 맥락에 맞는 판단을 점점 더 정확하게 합니다.
