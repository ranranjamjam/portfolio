Phase 0 유료 광고 캠페인

트래픽은 나오는데 리드가 0 — 전환 캠페인을 처음부터 재설계하다

영상 소재로 Meta 광고를 집행했더니 트래픽은 잘 나왔다. CTR 1.5~2.3%, 주간 유입 1,500명. 하지만 문의(리드)는 단 한 건도 없었다. "트래픽 ≠ 전환"이라는 문제를 직면하고, 리드를 만들기 위한 채널·소재·퍼널을 처음부터 재설계하기로 했다.

원인 진단

2

3채널 소액 테스트(Phase 0) 설계

본예산 300만원 투입 전, 35만원(1주)으로 3채널 동시 검증. Google RSA(검색 의도) + Meta 제품 소구점 + Meta 블로그 소구점. GDA는 소액 예산 분산 시 채널당 데이터 부족으로 제외 판단.

3

전환 특화 소재 6종 제작

페르소나 기반 소구점 3축 도출: 시간("아직도 링크드인 뒤지고 계세요?") / 품질("또 잘못 뽑으면 3개월 날립니다") / 비용("서치펌 수수료가 아깝지 않으세요?"). 블로그 콘텐츠 3편도 별도 광고 소재로 제작.

블로그 콘텐츠 마케팅

"콘텐츠는 있는데 검색으로 아무도 안 들어온다" — SEO 기반 구축부터 키워드 1위까지

PROBLEM

고객 획득의 64%가 인바운드(검색·블로그·문의)인데, 검색 유입 채널이 사실상 미작동. 콘텐츠는 외부 블로그에 흩어져 있고, 자사 블로그는 모든 페이지의 <title>이 동일하고 구조화 데이터 0건, 내부링크 0개 — SEO 기본기가 없는 상태였다. 콘텐츠 수정에도 매번 개발자 공수가 필요해 마케터가 독립적으로 실험할 수 없었다.

HYPOTHESIS

SEO 기반을 깔고(CMS 이관 + 기술 대응) + 유효 키워드를 선별해 콘텐츠를 집중하면, 검색 유입이 생길 것이다.

ACTION
1

SEO 기술 점검 → Ghost CMS 이관

HTML 기반 SEO/GEO 12항목 체크리스트를 작성하고 개발팀에 P0 요청(title 고유화, 내부링크 <a> 전환). WordPress/Ghost/직접 대응을 비교 검토한 끝에 Ghost CMS 채택 — 마케터가 직접 발행·수정할 수 있는 구조 확보.

2

키워드 분석 → 유효 키워드 15개 선정

구글 서치콘솔 + 키워드 플래너로 실제 유입 키워드를 분석. 클릭 발생 키워드 15개를 선별하고 "1위 유지 / CTR 보강 / 장기 보강" 3단계로 우선순위 분류. 트래픽용(HR 실무 팁) vs 전환용(서비스 연결) 콘텐츠 이원화 전략 수립.

3

4채널 배포 시스템 구축

Ghost 원본 → 네이버 블로그·브런치·링크드인·이오플래닛 배포 자동화. 네이버 Search Advisor 세팅 + 브런치 작가 등록까지 직접 완료. 외부 백링크 확보로 도메인 권위 성장 기반 마련.

LinkedIn 광고 캠페인

HR/CEO 타겟 분리 운영 + 소재 A/B 테스트 — B2B 광고 학습 주기 체득

PROBLEM

B2B SaaS 리드 확보를 위한 LinkedIn 광고. HR 담당자와 CEO/C-Level이라는 서로 다른 의사결정자에게 동시에 도달해야 하는 상황. B2B 광고는 D2C와 달리 노출→인지→재방문→리타겟팅→문의의 긴 학습 주기를 거쳐야 한다는 전제부터 확인이 필요했다.

ACTION
1

캠페인 이원화 구조 설계

캠페인 A(JD Creator, HR 인사담당자 타겟) + 캠페인 B(C-Level 인재 샘플, 경영진 타겟)로 분리 운영. 의사결정자별 메시지·CTA를 독립적으로 관리.

2

다양한 소재 A/B 테스트

강남언니·오호라·대기업·C레벨·CFO 등 다양한 소재 실험. "인재샘플 받아보기" vs "JD 컨설팅 받기" CTA 비교. CTR 분석→가설 수정→소재 확장 사이클 자체 운영.

3

소재 톤 벤치마크 수립

enforce vs soft 톤 비교 테스트. Lead Gen Form 최적화. 초기 CTR ~0.3%에서 소재 최적화를 통해 개선.

이벤터스 웨비나 마케팅

모객 54→137명, 참석율 77% — HOT/WARM/COLD 세분화 후속 관리까지

PROBLEM

초기 등록 54명에서 출발한 B2B 웨비나. 단순 모객이 아닌, "등록 → 참석 → 리드 너처링"까지 이어지는 체계가 필요했다. 참석 후에도 HOT/WARM/COLD에 따라 다르게 관리해야 세일즈로 이어진다는 가설.

ACTION
1

모객 캠페인 설계

이벤터스 플랫폼 활용 + LinkedIn·뉴스레터·블로그 크로스 프로모션. 초기 54명에서 최종 137명까지 확장.

2

참석 유도 메일 시퀀스

등록자 대상 D-7, D-3, D-1, D-Day 리마인드 시퀀스 설계. 참석율 77% 달성.

3

HOT/WARM/COLD 후속 관리

참석자를 3등급으로 세분화하여 후속 메일을 등급별로 차별화. COLD 대상에게는 발표자료 요약본을 별도 제작해 제공. 오픈카톡방 커뮤니티(37명) 운영으로 대화형 너처링.

GA/GTM 분석 인프라 구축

작동하지 않던 분석 환경을 처음부터 재설계 — 마케터가 직접 만든 데이터 인프라

PROBLEM

"분석 도구는 있는데 제대로 작동하지 않는" 상태. 이벤트가 수집되지 않고, 내부 트래픽이 필터링되지 않아 데이터가 ~35% 과대 추정되고 있었다. 데이터 기반 의사결정이 구조적으로 불가능한 환경. 엔지니어 지원 없이 마케터가 직접 해결해야 했다.

ACTION
1

기존 GA 문제 진단

이벤트 누락 항목 정리, 내부 트래픽 포함 여부 점검, 전환 퍼널 정의 부재 확인. 측정 도구 자체가 신뢰 불가 상태임을 리포팅.

2

GTM 신규 설치 + 이벤트 정의서 작성

Google Tag Manager 신규 설치. 전환 퍼널별로 generate_lead, form_start 등 이벤트 정의서 작성 후 GA4에 전면 재설정.

3

내부 트래픽 필터 + 3-stack 연동

내부 IP 트래픽 제거 (3월 말부터 적용). GA4 + GTM + PostHog 3개 도구 연동 체계 수립. 채널별(Organic/Paid/Social/Direct) 유입→전환 추적 시트 제작.

컴업 박람회 부스 운영

오프라인 이벤트 경험 0 — 부스 기획부터 리드 수집 체계까지 주도

PROBLEM

2025년 12월 컴업(COMEUP) 박람회 참가. 오프라인 이벤트 운영 경험이 없는 상태에서 부스 기획·운영·리드 수집 전반을 혼자 맡아야 했다.

ACTION
1

서비스 청사진(Service Blueprint)

방문객 관점의 고객 여정을 시각화. "부스 인지 → 관심 → 대화 → 리드 전환"의 각 접점에서 필요한 메시지·행동·자료를 사전 설계.

2

응대 스크립트 + 사은품 제작

현장 응대 스크립트 제작(업종별 변형 포함). 볼펜·에코백 등 사은품 제작·관리.

3

QR 리드 수집 체계

현장 QR 코드 기반 리드 수집 플로우 구축. 방문객 이탈 없이 정보를 받아 CRM으로 연결하는 체계 수립.

홈페이지 CRO 스프린트

광고로 풀리지 않은 5개월, LP에서 풀어낸 11일 — 진단부터 후속 사이클까지

PROBLEM

2025년 11월 15일 입사 시점, 회사는 디지털 인바운드 lead가 들어오지 않는 상태였다. 소개·지인·추천으로 굴러가고 있었지만, 반복 가능한 인바운드 채널이 없다는 게 진짜 문제였다.

인턴 기간을 거쳐 마케팅 본격 투입 후 1월부터 광고를 시작했다. LinkedIn 단발성으로 가설 검증 → 4월 Phase 0(Meta + Google RSA) 본격 확장 → 영상 광고까지 추가해 CTR 평균 2∼3% 수준까지 끌어올렸다. 2월부터는 방치돼 있던 블로그를 Ghost CMS로 이관해 SEO도 병행했다.

4개월간 광고 약 490만원, SEO 3개월 병행. 그래도 lead 0건이었다. 클릭은 들어오는데 폼까지 가지 않았다. 채널·소재·예산을 다 시도하고도 0이라는 건, 변수는 LP라는 뜻이었다. B2B 고관여 상품(선불 100만원)은 광고 클릭만으로 결정되지 않는다는 결론에 도달했다.
